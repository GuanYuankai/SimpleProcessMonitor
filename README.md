### Dependencies

* CMAKE VERSION >= 3.14 -> [link](https://cmake.org/install/)
* Lib Qt 5 -> `sudo apt-get install qt5-default`
* Lib Procps -> `sudo apt-get install libprocps-dev`

### Compiling

`git clone https://github.com/y9luiz/SimpleProcessMonitor`

`mkdir build`

`cd build`

`cmake ..`

`make`

### Runing 

`./run`
